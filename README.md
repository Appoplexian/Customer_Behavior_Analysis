# Customer_Behavior_Analysis
End-to-end Data Analytics project transforming raw retail transactional data into actionable business strategies. Features Python EDA, SQL-based customer segmentation, and an interactive Power BI dashboard to optimize marketing spend and subscription growth.

# Customer Shopping Behavior & Revenue Optimization

## 📌 Project Overview
This project analyzes a dataset of **3,900 customer transactions** to identify patterns in spending, subscription habits, and demographic preferences. The goal is to provide data-driven recommendations to increase customer lifetime value and optimize marketing efforts.

## 🛠️ Tech Stack
- **Data Cleaning & EDA:** Python (Pandas, Matplotlib, Seaborn)
- **Database Management:** PostgreSQL (Complex Joins & Aggregations)
- **Data Visualization:** Power BI (Interactive Executive Dashboard)

## 📊 Key Business Insights
- **Revenue Driver:** Male customers contributed **67.7% ($157k)** of total revenue, indicating a strong core demographic.
- **Subscription Gap:** Only **27%** of customers are currently subscribers, despite similar spending habits to non-subscribers—representing a major conversion opportunity.
- **Shipping Sensitivity:** Customers using **Express Shipping** have a higher average transaction value ($60.48), suggesting that premium shipping tiers could be a lever for growth.
- **Discount Impact:** Identified that certain categories like **Hats** are highly dependent on discounts (50% application rate).

## 🚀 Business Recommendations
1. **Loyalty Program:** Launch a tiered loyalty system targeting the "Middle-aged" segment (the highest revenue group) to move them into the 'Loyal' segment.
2. **Subscription Push:** Offer "Free Express Shipping" as a subscriber-only benefit to increase the 27% subscription rate.
3. **Seasonal Inventory:** Adjust stock levels for high-performing categories identified in the Seasonal Analysis section of the report.
